    • Using these links have a variety of expected results, stemming from User Entitlement Status.
    These have been outlined beneath each link provided.
    To put it simply, the app should handle these deeplinks the same way it handles regular navigation of these items.

    • Non E+ Featured Page Event
      • Not Logged In, Demo Unused - Feed will launch in Demo Mode with the countdown beginning
      • Not Logged In, Demo Expire - The "Provider Login Flow" attempt to begin
      • Logged In - Feed opens
    • ESPN+ Event: Pick Feed
      • Not Logged In - Feed Picker appears
      • Logged In - "Best Estimate" feed launches. Feed Selector is accessable in the video controller
    • ESPN+ Event: Direct to Feed
      • Not Logged In - Paywall appears
      • Logged In - Video Playback begins with the designated broadcast. Feed selector is NOT available this way (unsure if issue, stems from source ESPN code).
    • ESPN+ Event: Direct to Feed (Another example)
      • Not Logged In - Paywall appears
      • Logged In - Video Playback begins with the designated broadcast. Feed selector is NOT available this way (unsure if issue, stems from source ESPN code).
    • ESPN Channel: Sportscenter
      • Not Logged In - Provider Login flow
      • Logged In - Video Playback begins with the designated broadcast. Feed selector is NOT available this way (unsure if issue, stems from source ESPN code).
    • ESPNU Channel: Miracles on the Plains
      • Not Logged In - Provider Login flow
      • Logged In - Video Playback begins with the designated broadcast. Feed selector is NOT available this way (unsure if issue, stems from source ESPN code).
